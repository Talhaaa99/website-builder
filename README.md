# Website Builder

## Introduction
This project is a simplified version of a website builder that allows users to create, edit, and preview a basic website using a drag-and-drop interface.

## Technologies Used
- Next.js
- React.js
- TypeScript
- Tailwind CSS
- react-dnd

## Features
- Drag-and-Drop Interface
- Element Properties Editing
- Page Management
- Template System
- Preview Mode
- Responsive Design
